Image Processing Lab Excercises

Image Processing Lab7: Noise Models

Exercise 1: Gaussian Noise Model (Practice)
Generate Gaussian noise of 300x500 pixels, using mean u=127 and standard deviation sd=50. Display this noise image and its histogram. Note that the Gaussian distribution is also called the Normal distribution.

Exercise 2: Rayleigh Noise Model (Practice)
Repeat Ex1 using noise following the Rayleigh Distribution. Note that Gaussian distribution had two parameters - mean and standard distribution. For Rayleigh noise, MATLAB uses only one parameter, b. Use b=127 for this problem. Also, note that MATLAB's build-in model is slightly different than what you studied in the class, which used two parameters.

Exercise 3: Erlang (Gamma) Noise Model (Practice)
Repeat Ex1 using noise following the Gamma Distribution.

Exercise 4: Exponential Noise Model (Practice)
Repeat Ex1 using noise following the Exponential Distribution.

Exercise 5: Uniform Noise Model (Practice)
Repeat Ex1 using noise following the Uniform Distribution.

Exercise 6: Salt and pepper (Impulsive) Noise Model (Practice)
Repeat Ex1 using noise following the Salt and pepper model. (Note: A straightforward way to generate this noise is to randomly set some pixels to 255 and some others to zero.)

Exercise 7: Salt and pepper (Impulsive) Noise Model (Bonus)
Repeat Ex6 using a different strategy to generate the Salt and pepper noise. Be innovative.

Exercise 8. (Marked)
Read the cameraman image. Add to it the Gaussian noise of mean u=10 and standard deviation sd=10.
(a) Show the histograms of the original and the noisy images side-by-side. Can you identify the type of noise from the histogram? Explain your answer.
(b) Repeat the above for u=50 and sd=10 and answer the same question as in (a).
(c) Repeat the above for u=10 and sd=50 and answer the same question as in (a).


Exercise 9. (Marked) Read the cameraman image. Add salt and pepper noise, each using a probability p=0.05.
(a) Show the histograms of the original and the noisy images side-by-side. Can you identify the type of noise from the histogram? Explain your answer.
(b) Repeat the above for p=0.2 and explain the change you see in the noisy image and the histogram compared to (a).
