Exercise 1: Zero-Padding (Practice)
In image filtering (convolution of an image with a filter kernel), when the center of the kernel is aligned with a boundary pixel, some kernel elements go outside the image boundary and do not have any corresponding pixels to multiply. Therefore we need to add dummy rows and columns on each side of the image. A simple way is to add rows/columns of zero value on all four sides. This is called zero-padding. Generate a 6x7 test image of all ones. Assuming that you want to filter it using a 5x7 kernel, do the required zero-padding to the test image.
